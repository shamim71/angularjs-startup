# F1FeederApp-Part2

## Getting Started

All you need to do is to clone this repository,


```
git clone https://github.com/raonibr/f1feeder-part2
cd f1feeder-part2
```

Then, install all the dependencies:

```
npm install
bower install
```

Then, run the Application:

```
grunt serve
```

And to run the tests:

```
grunt test
```
